# App Branding Assets

Replace these files before generating the Android APK:

- `icon.png` (1024x1024 PNG)
- `adaptive-icon.png` (1024x1024 PNG, foreground only)
- `splash-icon.png` (recommended large PNG, centered logo)
- `favicon.png` (48x48 or 64x64 PNG, optional for web)

Current files are placeholders copied from Expo defaults.
